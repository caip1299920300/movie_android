# movie_android
安卓网络视频播放器

本项目借鉴了GitHub中：https://github.com/Jzvd/JZVideo



## 1.本项目的特色

​		1.在借鉴的项目上提取了网络视频播放的部分代码，使大家可以快速的弄出这一部分的功能。

​		2.只需要在项目MainActivity.java中，修改网络地址即可实现播放自己的功能
